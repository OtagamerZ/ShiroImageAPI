# ShiroImageAPI
## How to start it?
- Clone this repository
- Run `gradlew bootRun`
- Done!

## How to add new GIFs?
- Create a new folder inside resources (or choose one of the current)
- Drag the file inside it
- Run `check_sizes.sh`
  - If adding GIFs to an existing folder, remove `.checked` file
- Done!
